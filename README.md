# Test Assignment
A simple app with fastapi, async, redis, pg

## Preconditions:

- Python 3

## Clone the project

```
git clone https://github.com/victor-kalinin/test-assignment.git
```

## Run with docker

### Run server

```
docker-compose up -d --build
```

### Open in browser

```
http://127.0.0.1:3000
```

## API documentation (provided by Swagger UI)

```
http://127.0.0.1:3000/docs
```
